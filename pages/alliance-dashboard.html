Alliance Dashboard

Live war snapshots, chains, and a shared weapon vault synced in the cloud. The alliance creator can add any faction IDs to the roster; each faction’s Leader / Co-leader can see every alliance they’ve been added to and hide any from their personal list. Creator + leader tools

My alliances

Shown when your Torn account is Leader or Co-leader of your faction and that faction was added to an alliance. Use Open to load intel and vault for that alliance; Settings opens the alliance panel (ID, roster, add faction, etc.); Hide only hides the card in this browser.

Alliance Intel

Auto

This page uses the API key from the sidebar to load live data from Torn. Refresh intel updates war scores, chains, and whether each faction’s territory tiles are safe or at war (open a card’s territory list to see). Refresh territory uploads your faction’s map from Torn so everyone in the alliance can see it on the cards. Extra tile letters you add by hand are removed when that faction uses Refresh territory again. Auto refreshes on a timer; very short intervals may get rate-limited by Torn.

Shared weapon vault

Stored in Firestore. Any member of an allied faction can add or edit entries. Paste item text from Torn in the modal and use Extract to fill stats.
