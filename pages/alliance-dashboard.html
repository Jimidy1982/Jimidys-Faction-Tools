Alliance Dashboard

Live war snapshots, chains, and a shared weapon vault synced in the cloud. The alliance creator can add any faction IDs to the roster; each faction’s Leader / Co-leader can see every alliance they’ve been added to and hide any from their personal list. Creator + leader tools

My alliances

Shown when your Torn account is Leader or Co-leader of your faction and that faction was added to an alliance. Use Open to load intel and vault for that alliance; Settings opens the alliance panel (ID, roster, add faction, etc.); Hide only hides the card in this browser.

Alliance Intel

Auto

Uses your sidebar API key against Torn (ranked war list + chain per faction, plus torn?selections=territorywars on each Refresh intel). Use Refresh territory to push your faction’s tiles from Torn into this alliance (others then see them on the cards). Tile letters show Safe or At War in each card’s dropdown; manual tile codes clear when that faction refreshes territory from Torn. Turn Auto on to poll on a schedule; short intervals may hit Torn rate limits.

Shared weapon vault

Stored in Firestore. Any member of an allied faction can add or edit entries so everyone knows where gear is (armoury, loaned to player, etc.).

Create or join an alliance

Join with an ID someone shared, or create a new alliance (your faction is added first; you must be Leader or Co-leader).

Join existing

Paste the alliance ID, then load. You can still use Open under My alliances when you’re listed there.

Create new

Starts a new alliance document and adds your current faction to the roster.

Alliance settings

Load an alliance to use this panel.

Alliance ID

Share this ID so other faction leaders can load the alliance here or under My alliances once their faction is on the roster.

Faction roster

You are not the alliance creator. You can copy the ID, view the roster, and add your own faction (if you are Leader / Co-leader). Renaming, removing factions, and directory repair are creator-only.

Territory text note

Use Refresh territory on the Alliance Intel panel to sync tiles from Torn for your faction. If someone cannot sync, the alliance creator or any roster Leader / Co-leader may use Edit in the roster above for a plain-text note; when that faction later refreshes territory from Torn, the note is replaced.

Add faction

Alliance creator may add any faction ID. Other Leader / Co-leader accounts (with their faction’s API key in the sidebar) may add only their own faction.

Rename alliance

If this alliance existed before the “My alliances” directory, click once to sync faction rows to the directory.
