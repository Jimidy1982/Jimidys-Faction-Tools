Vault Checker

See what's in your faction vault and who has loaned items. Borrower names come from the API when available; when the API omits them (common for “loan yourself”), we backfill from recent faction armory news. We flag players who have 2+ weapons of the same type (Primary, Secondary, Melee) or multiple armour in the same slot.

API key selections (add these on your custom key if anything fails): Faction → weapons & armour; User → profile & basic; Faction (v2) → members; Faction news → armory (for borrower backfill). The same key can power your other tools.
